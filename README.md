# Sismos-Chl

## Pre-Requisites
```
- Yarn version 1.17.3
- Node version 12.10.0
- Vue version @vue/cli 4.2.2
- Ionic version 5.4.16
```

## For run the proyect you need install dependencies 
```
yarn install
```

### then you run the server with
```
yarn serve
```

### the routes for aplication are 
```
http://localhost:8080/ 
```
